## Flappy bird Gamerella 2025
This project is a simple Flappy Bird-style game developed in Unity using C#. The goal of this work was to explore the Unity engine for the first time and gain a basic understanding of C# scripting within a game development context.

The project was completed as part of a solo game jam and was primarily guided by an online tutorial, focusing on learning core mechanics such as player movement, collision detection, and basic game loop structure. Through this process, I gained initial hands-on experience with Unity’s workflow and how scripts interact with game objects to create simple interactive gameplay.
